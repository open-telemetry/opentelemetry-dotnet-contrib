# OpenTelemetry StackExchange.Redis Instrumentation Benchmarks

Navigate to the `./test/OpenTelemetry.Instrumentation.StackExchangeRedis.Benchmarks`
directory and run the following command:

```sh
dotnet run --configuration Release --framework net11.0
```

Then choose the benchmark class that you want to run by entering the
desired option from the list of options shown in the terminal window.
