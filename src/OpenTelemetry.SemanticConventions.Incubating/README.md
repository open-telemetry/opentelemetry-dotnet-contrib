# Incubating Semantic Conventions for OpenTelemetry .NET

This project contains generated development, alpha, beta, release candidate, and
deprecated attributes for the [Semantic Conventions](https://github.com/open-telemetry/semantic-conventions)
defined by the OpenTelemetry specification.

## Installation

```shell
dotnet add package OpenTelemetry.SemanticConventions.Incubating --prerelease
```

Stable attributes are generated in the `OpenTelemetry.SemanticConventions`
package.
