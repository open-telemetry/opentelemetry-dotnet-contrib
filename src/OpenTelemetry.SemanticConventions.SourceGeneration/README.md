# OpenTelemetry.SemanticConventions.SourceGeneration

Roslyn source generator for OpenTelemetry semantic-conventions attribute keys.

## Use

```csharp
using OpenTelemetry.SemanticConventions.SourceGeneration;

[SemanticConventionAttributes("disk")]
internal static partial class DiskAttributes;

// Generated:
//   public const string AttributeDiskIoDirection = "disk.io.direction";
//   public static class DiskIoDirectionValues
//   {
//       public const string Read  = "read";
//       public const string Write = "write";
//   }
```

The marker attribute is itself generated (via `RegisterPostInitializationOutput`);
no runtime dependency is added by consuming this package.

## Versioning

Tracks `open-telemetry/semantic-conventions` `v1.41.0` (commit
`e018fe6f91862f5ed63c082f87697cddac596784`), embedded via Weaver `v0.23.0`.
The embedded registry is regenerated by `scripts/generate.sh`.

The generated member shape matches upstream
`open-telemetry/opentelemetry-dotnet-contrib`
`OpenTelemetry.SemanticConventions.SourceGeneration` byte-for-byte (modulo
the namespace token), enforced by the byte-identity audit gate.
